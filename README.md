# DOCUMENTACIÓN TÉCNICA
Proyecto: Portal de Turismo – Huellas Argentinas
Estructura
El sitio fue desarrollado utilizando HTML5 con etiquetas semánticas como header, nav, main, section, article y footer.
Se organizó el contenido en diferentes páginas: Inicio, Destinos, Agencias, Precios, Blog y Contacto.
# Diseño y Estilos
Se utilizó CSS3 externo para los estilos.
Se implementaron CSS Grid, Flexbox, variables CSS y diseño responsive mediante media queries.
# Funcionalidades
Menú con mega-menú desplegable.
Tarjetas interactivas con efecto flip.
Filtros por categoría usando :checked.
Sistema de rating con estrellas animadas.
Animaciones con keyframes y scroll suave.
# Técnicas Avanzadas
Uso de variables CSS.
Clip-path en imágenes.
Modo oscuro con JavaScript mínimo.
Micro-interacciones en botones y enlaces.
# Accesibilidad
Uso de aria-label.
Estados de focus visibles.
Buen contraste de colores.
